Create Third Places was started by Bharat Ponnaluri to promote accessible in person in person spaces. It is inspired by the organization Strong Towns, which empowers people to improve communities through bottom up change. It is also based on the ideas of sociologist Ray Oldenberg who coined the term "Third Place". Create Third Places is a non-commercial open source entity.

Current activities

Bharat hosts public board game events on weekend afternoons in DC and regularly attends other public events in third places. He is also a member of Strong Towns DMV, a Strong Towns local conversation group.

Create Third Places also created the following projects to facilitate in person interaction through public spaces.

  • Places.js: Framework for building websites promoting in person interaction.
  • Want to play board games and make new friends in the DC area. Find dates, times and locations, go to dmvboardgames.com
  • Map listing: A work in progress page with the ability to create and edit a list of locations and have them displayed as points on a map.
  • Toaster Trouble board game
Create Third Places also working on software to support interaction in third places. See the software page for more informaiton.
