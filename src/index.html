Create Third Places was created by Bharat Ponnaluri to promote accessible offline public spaces. It is inspired by the organization Strong Towns, which empowers people to improve towns and neighborhoods through bottom up change. Create Third Places is a non-commercial open source entity.

Current projects

  • Places.js framework for building websites promoting in person interaction
  • dmvboardgames.com: The goal of dmvboardgames.com is to promote human connection through board games as a way of increasing interest in public spaces. It uses the places.js framework.
  • Map listing: This is a work in progress page with the ability to create and edit a list of locations and have them displayed as points on a map.
  • Toaster Trouble board game
