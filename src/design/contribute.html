Contributing to Create Third Places design

If you are interested in contributing to one of these areas listed below, or have other design suggestions, email gulu@createthirdplaces.org.

  • Updating the design system for Create Third Places.
  • Improving look and feel of links.
  • Make sure website is acessible.
  • Web design help to make the website UI look better.
  • Recommending a color scheme for UI elements.
  • Recommending a set of fonts for text.

Create Third Places has a strict no generative AI policy and createthirdplaces.com blocks automated scraping.
