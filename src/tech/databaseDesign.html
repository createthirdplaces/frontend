Database Design

This is an example of how data for the decentralized architecture can be stored in a relational database, and it exists as a starting point for developers. It is not necessary to precisely follow this example as interaction between different websites is defined by API endpoints.

Database tables for aggregator websites.

group_aggregation

  • group_uuid: UUID of group.
  • local_website_url: API url for group website group is from. This value must be present in the local_websites label.

local_websites

  • local_website_urlAPI url for group website.
Database tables for group websites

aggregators

  • aggregator_url(Primary key): URL of aggregator website.

events

  • group_id(Foreign key): Represents the id of the group the event is for.

groups

  • uuid(Primary key): A randomly generated id that is a unique identifier for a group.
