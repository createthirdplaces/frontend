Overview

Places.js is a lightweight human-centric Javascript framework for creating interactive websites. All code was written by hand without LLM assistance from tools such as ChatGPT or Claude Code, and LLMs will not be used for future changes. Source code is here

Places.js is designed to be an alternative to Lit and React.
